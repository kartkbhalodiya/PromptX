REST API

Simple HTTP endpoints that work with any language

POST /api/enhance

Enhance a prompt with AI-powered improvements

Request

curl -X POST https://api.promptx.dev/api/enhance \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"prompt": "Write a blog post about AI"}'

Response

{
  "success": true,
  "enhanced": "Write a comprehensive, SEO-optimized blog post...",
  "original_score": 4.5,
  "enhanced_score": 9.2,
  "improvement": 4.7,
  "model": "gemini-2.0-flash"
}
